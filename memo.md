# Software Security 学习

本目录用于整理 software security 的路线图、知识笔记、可复现实验和参考材料。部分内容参考 Software Security 课程(Michael Hicks, UMD / Coursera)。

## 目录结构

- `PLAN.md` — 推荐路线图
- `reports/` — 实验记录与复盘
- `reports/TEMPLATE.md` — 复盘模板,新建记录时复制
- `labs/` — 动手实验代码、漏洞复现、笔记
- `notes/` — 知识笔记
- `original labs/` — UMD / Coursera archived labs

## 工作方式

- 每个主题尽量配一个**可运行的动手产出**(实验 / 漏洞复现 / fuzzing demo)。
- 重要实验和踩坑记录放在 `reports/` 下,方便复盘。
- 用 git 跟踪代码、文档和实验环境说明。
